# Colocation Management System

A web application that helps manage shared housing (colocation), including members, expenses, and payments.

## 🚀 Features
- Manage colocations (create, activate, deactivate)
- Manage members in a colocation
- Track payments made by each member
- Calculate total paid by a member
- Compare paid amount with the theoretical average share

## 🛠️ Technologies Used
- SQL (MySQL / PostgreSQL)
- Backend: (Java / PHP / Laravel / Spring Boot) ← update this
